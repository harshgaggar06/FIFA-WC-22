# FIFA World Cup 2022 Analytics Dashboard

An interactive Power BI dashboard that transforms FIFA World Cup 2022 data into actionable insights. This project analyzes team performance, player statistics, match outcomes, possession trends, goals, assists, disciplinary records, and tournament progression through dynamic visualizations and interactive filters.

## Tools Used
- Power BI
- Power Query
- DAX
- Microsoft Excel

## Dashboard Highlights
- Tournament Overview
- Team Performance Analysis
- Player Statistics
- Match Analysis
- Goal & Assist Leaderboards
- Possession & Tactical Insights
- Interactive Filters and Drill-through
